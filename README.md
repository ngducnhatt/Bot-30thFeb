## Tính năng (update version 1.4)
* Câu lệnh
    - Admin (admin only)
        - `/clear`: Xóa tin nhắn
        - `/notification`: Tạo thông báo từ bot đến kênh
    - Tính năng
        - `/qr`: Tạo mã QR
        - `/random`: Số ngẫu nhiên
        - `/short-url`: Rút ngắn liên kết url
        - `/weather`: Thời tiết của thành phố
        - `/ping`: Kiểm tra tốc độ
        - `/contact`: Thông tin của tôi
